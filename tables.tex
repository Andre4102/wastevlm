\caption{%
  Zero-shot multi-label classification with open chain-of-thought (CoT)
  prompting on DroneWaste paper-10 (DW, 1504 test images) and
  AerialWaste m2/m4 (664 test images each).
  Micro\,F1 reported; macro\,F1 in parentheses.
  \textit{kw-bag}: keyword-bag parser on Turn-2 output.
  \textit{v1}: Turn-2 examples hardcoded to DW vocabulary (biased).
  \textit{v2}: Turn-2 examples drawn from each dataset's synonym list;
  AW\,m4 ``Other waste'' bag also expanded with hedging phrases.
  \textit{Judge T1/T2}: LLM judge (Claude Haiku\,4.5) applied to the
  free visual description (Turn\,1) or to the classification output
  (Turn\,2).  Bold: best result per column.
  CLIP zero-shot included as reference.
  On AW\,m2, CLIP retains the best score because its tag embeddings were
  designed for the dominant categories (Bulky items, Containers) and
  because embedding-based retrieval is robust to the coarser 100\,m GSD
  of this split, whereas VLMs often produce vague shape descriptions at
  that resolution.
  On AW\,m2 the kw-bag v2 parser matches or outperforms the LLM judge:
  the v2 Turn-2 prompt anchors the VLM to dataset-specific vocabulary,
  so the output already names the category explicitly and substring
  matching is sufficient.
  On DW the pattern reverses — 44\,\% of Turn-2 responses are ``none''
  and the remaining outputs are hedged or paraphrased, so the LLM judge
  extracts substantially more signal than the keyword bag.
}
